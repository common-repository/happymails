== Description ==

HappyMails is a simple plugin that automatically sends happy birthday emails to users on their birthdays. You can use this plugin to make your users or customers happier and increase their loyalty.

The plugin is made and maintained by [FitWP](http://fitwp.com)

== Installation ==

1. Go to *Plugins* and search for **HappyMails**
2. Click **Install** button to install the plugin
3. Click **Activate** to activate the plugin

Then in user's profile page will have a new input for birthday. You need to add user birthday manually.

To compose the email, please go to **HappyMails** menu.
